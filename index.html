١. البرمجة يعني إيه؟

إزاي الكمبيوتر بيفكر؟

خلينا نتخيل إن الكمبيوتر زايد شوية... يعني لو قولتله "روح اشترلي خبز" مش هيعرف يعمل حاجة. بس لو قلناله:

  1. امشي 10 خطوات لليمين
  2. لف شمال
  3. امشي لحد ما توصل للسوبرماركت
  4. ادفع 5 جنيه
  5. خد رغيف العيش
  6. ارجع بالظبط نفس الطريق

دي بالظبط فكرة البرمجة! إحنا بنقول للكمبيوتر بالظبط إزاي يعمل الحاجة خطوة خطوة.

س١: لو عايز الكمبيوتر يحسب ناتج جمع ٥ + ٣، إيه الخطوات اللي ممكن نكتبها؟

إيه اللي يخلي بايثون مميزة؟

بايثون زي العربي الفصيح.. سهلة ومفهومة من أول مرة! كود بايثون بيقرأه أي حد تقريباً عارف إنجليزي:

if age > 18:
    print("أنت بالغ يا معلم!")

مقارنة بلغات تانية زي الجافا اللي محتاجة كتير جداً عشان تعمل حاجة بسيطة:

public class Main {
    public static void main(String[] args) {
        int age = 20;
        if (age > 18) {
            System.out.println("أنت بالغ يا معلم!");
        }
    }
}
س٢: إيه اللي يخلي بايثون مناسبة للمبتدئين؟

نصيحة أخيرة

متخليش كلمة "برمجة" تخوفك! الفكرة كلها إنك بتعلم الكمبيوتر يعمل حاجات بالترتيب اللي انت عايزه. زي ما بتعلم طفل صغير يعمل حاجة خطوة بخطوة.

٢. يا ترى بايثون ده ثعبان؟

قصة بايثون

بايثون مسمية على اسم مسلسل بريطاني قديم اسمه "Monty Python's Flying Circus" مش الثعبان! المبتكر جايدو فان روسوم كان بيحب المسلسل دا فسمى اللغة على اسمه.

بايثون ظهرت سنة 1991 (يعني تقريباً نفس عمر أغنية "كان عندي أمل") ومن ساعتها وهي بتتطور وتنتشر.

س١: إيه سبب تسمية لغة بايثون بهذا الاسم؟

ليه نتعلم بايثون؟

بايثون سهلة جداً للمبتدئين، وكمان قوية جداً. يعني ممكن تستخدمها في:

  • برمجة المواقع (مثل Instagram و Pinterest)
  • الذكاء الاصطناعي وتعلم الآلة
  • تحليل البيانات والإحصاء
  • ألعاب الكمبيوتر
  • أتمتة المهام المملة (مثل عمل نسخ احتياطي للصور)
# بايثون سهلة قراية حتى لو مبتدئ
print("أهلاً بالعالم!")
for i in range(5):
    print("بايثون رائعة!")

إزاي تنزل بايثون على جهازك؟

١. روح على موقع بايثون الرسمي

٢. نزل النسخة المناسبة لجهازك (هتلاقيها في أول الصفحة)

٣. شغل الملف اللي نزلته واتبع الخطوات (كلها Next لحد ما تخلص)

٤. افتح CMD أو Terminal واكتب python --version عشان تتأكد إنها شغالة

س٢: إيه أول حاجة تعملها عشان تتأكد إن بايثون شغالة على جهازك؟

٣. الطباعة على الشاشة

أول برنامج ليك في بايثون

أول حاجة هنتعلمها هي الطباعة على الشاشة باستخدام print(). دي أسهل حاجة في بايثون!

# دي دالة الطباعة
print("أهلاً بك في بايثون بالعربي!")

جرب الكود دا في IDLE أو في أي محرر نصوص وحفظه بامتداد .py

إزاي تطبع أكتر من سطر؟

ممكن تستخدم دالة print() أكتر من مرة، أو تستخدم ثلاث علامات تنصيص """

# الطريقة الأولى
print("السلام عليكم")
print("أهلاً وسهلاً بيك")
print("في بايثون بالعربي")

# الطريقة الثانية
print("""السلام عليكم
أهلاً وسهلاً بيك
في بايثون بالعربي""")
س١: إزاي تطبع جملة من سطرين باستخدام دالة print() مرة واحدة؟

٤. المتغيرات بقى

المتغيرات في الحياة

المتغيرات زي العلب اللي بنحط فيها حاجات. مثلاً:

  • علبة اسمها "فلوسي" فيها 50 جنيه
  • علبة اسمها "عمرك" فيها 20 سنة
  • علبة اسمها "اسمك" فيها "أحمد"

في بايثون، بنعمل نفس الحاجة بالظبط:

# تعريف متغيرات
money = 50 # جنيه
age = 20 # سنة
name = "أحمد" # اسم

إزاي نستخدم المتغيرات؟

بعد ما عرفنا المتغيرات، ممكن نستخدمها في أي حتة:

print("أهلاً", name)
print("عندك", money, "جنيه في جيبك")
print("العمر المثالي للبرمجة هو", age)
س١: إيه الناتج الصحيح للكود الآتي؟
x = 5
y = 2
print(x + y)

نصيحة مهمة

اختار أسماء واضحة للمتغيرات. مثلاً student_name أحسن من sn عشان لما ترجع للكود بعد شهر تفهمه!

٥. أنواع البيانات - نوعك إيه؟

أنواع البيانات الأساسية

في بايثون فيه أنواع كتير للبيانات، لكن دلوقتي هنتعلم الأساسيات:

أعداد صحيحة
أعداد عشرية
نصوص
قيم منطقية

الأعداد الصحيحة (int) زي: ١، ٥، -٣، ٠

age = 25 # عدد صحيح
students = 30 # عدد صحيح

الأعداد العشرية (float) زي: ٣.١٤، ٢.٥، -٠.٥

pi = 3.14 # عدد عشري
temperature = 36.7 # عدد عشري

النصوص (str) أي حاجة بين علامتي تنصيص: "أحمد"، "Hello"، "١٢٣"

name = "أحمد" # نص
phone = "0123456789" # نص (ولو كان أرقام)

القيم المنطقية (bool) بس ممكن تكون True أو False

is_raining = True # نعم
has_money = False # لا
س١: إيه نوع البيانات في الكود الآتي؟
price = "25.99"

إزاي أعرف نوع المتغير؟

استخدم دالة type() عشان تعرف نوع أي متغير:

print(type(10)) # <class 'int'>
print(type(3.14)) # <class 'float'>
print(type("hello")) # <class 'str'>
print(type(True)) # <class 'bool'>

٦. العمليات الحسابية... بس بالحب

العمليات الأساسية

بايثون بتعرف كل العمليات الحسابية الأساسية:

العملية الرمز مثال النتيجة
جمع + 5 + 3 8
طرح - 10 - 2 8
ضرب * 4 * 3 12
قسمة / 15 / 3 5.0
باقي القسمة % 14 % 3 2
أس ** 2 ** 3 8
س١: إيه نتيجة العملية الحسابية الآتية؟
15 % 4

نصيحة عملية

استخدم الأقواس () عشان تحدد أولويات العمليات الحسابية. مثلاً:

3 + 5 * 2 # النتيجة 13 (الضرب أولاً)
(3 + 5) * 2 # النتيجة 16 (الجمع أولاً)

٧. الجمل الشرطية - لو و إلا

اتخاذ القرارات في بايثون

الجمل الشرطية بتخلي البرنامج يتخذ قرار بناءً على شرط معين. زي لما تقول:

"لو الجو نازل مطر، خد الشتوية معاك، إلا خدها برضه عشان متتغرّبش!"

في بايثون بنكتبها كدا:

if is_raining:
    print("خد الشتوية معاك")
else:
    print("خدها برضه عشان متتغرّبش!")
س١: إيه الناتج الصحيح للكود الآتي؟
age = 20
if age >= 18:
    print("مسموح لك بالدخول")
else:
    print("ممنوع الدخول")

elif - وإلا لو

لو عنداك أكتر من شرط، استخدم elif (إختصار لـ else if):

grade = 85

if grade >= 90:
    print("امتياز")
elif grade >= 80:
    print("جيد جداً")
elif grade >= 70:
    print("جيد")
else:
    print("راسب")

٨. الحلقات - بنلف كتير ليه؟

لوب for - للدوران عدد معين من المرات

لما نعرف عدد المرات اللي عايزين نكرر فيها الكود، نستخدم for:

# اطبع الأعداد من 1 ل 5
for i in range(1, 6):
    print(i)

الكود دا هيطبع:

1
2
3
4
5
س١: إزاي تطبع "أحب بايثون" 3 مرات باستخدام for loop؟

لوب while - للدوران لحد ما يتحقق شرط

لما مش عارف عدد المرات بالظبط، لكن عارف الشرط اللي يوقف الدوران، استخدم while:

count = 1
while count <= 5:
    print(count)
    count += 1

نصيحة مهمة

خلي بالك من الـ infinite loops (الحلقات اللا نهائية)! لازم يكون في شرط يوقف اللوب، وإلا هيكمل لحد ما الجهاز يقع!

٩. الدوال - المطبخ بتاع الكود

إزاي تعرف دالة؟

الدالة (function) زي الوصفة في المطبخ.. بتجمع مجموعة من الخطوات في حاجة واحدة. تعريف الدالة بيبدأ بـ def:

def greet():
    print("أهلاً وسهلاً!")
    print("نورتنا اليوم!")

# استدعاء الدالة
greet()
س١: إيه الناتج الصحيح للكود الآتي؟
def say_hello():
    print("Hello")
    print("World")

say_hello()

الدوال مع باراميترات

ممكن نمرر قيم للدوال عشان تتصرف بناءً عليها:

def greet(name):
    print(f"أهلاً {name}! نورتنا اليوم!")

greet("أحمد")
greet("مريم")

١٠. القوائم - شنطة فيها حاجات

إزاي تعرف قائمة؟

القوائم (lists) في بايثون زي الشنطة فيها أكتر من حاجة. ممكن تحتوي على أي نوع بيانات:

# قائمة أسماء
names = ["أحمد", "مريم", "خالد"]

# قائمة أرقام
numbers = [1, 3, 5, 7]

# قائمة مختلطة
mixed = [1, "أحمد", True, 3.14]

إزاي تصل لعنصر في القائمة؟

كل عنصر في القائمة ليه رقم index بيبدأ من صفر:

names = ["أحمد", "مريم", "خالد"]

print(names[0]) # أحمد
print(names[1]) # مريم
print(names[2]) # خالد
س١: إيه الناتج الصحيح للكود الآتي؟
nums = [10, 20, 30, 40]
print(nums[2])

التعديل على القوائم

ممكن نضيف، نعدل، أو نحذف عناصر من القائمة:

# قائمة التسوق
shopping = ["خبز", "لبن", "بيض"]

# إضافة عنصر
shopping.append("جبنة")

# تعديل عنصر
shopping[1] = "لبن رائب"

# حذف عنصر
del shopping[2]

print(shopping) # ['خبز', 'لبن رائب', 'جبنة']

نصيحة عملية

استخدم len() عشان تعرف عدد العناصر في القائمة:

names = ["أحمد", "مريم", "خالد"]
print(len(names)) # 3

🎉 مشروعك الأول

اختر مشروعك

خلينا نطبق كل اللي تعلمناه في مشروع بسيط. اختر واحد من المشاريع دي:

تخمين العدد
آلة حاسبة
قائمة مهام

لعبة تخمين العدد

الكمبيوتر هيفكر في عدد عشوائي بين 1 و 20، وأنت بتخمنه لحد ما توصل للعدد الصحيح.

import random

# اختيار عدد عشوائي
secret_number = random.randint(1, 20)
attempts = 0

while True:
    guess = int(input("خمن العدد بين 1 و 20: "))
    attempts += 1

    if guess == secret_number:
        print(f"أحسنت! لقد وجدت العدد في {attempts} محاولات")
        break
    elif guess < secret_number:
        print("العدد أكبر من كدا")
    else:
        print("العدد أصغر من كدا")

آلة حاسبة متطورة

آلة حاسبة بتاخد من المستخدم عملية حسابية كاملة (مثل 5 + 3) وتحسب النتيجة.

while True:
    expression = input("ادخل العملية الحسابية (أو 'exit' للخروج): ")
    if expression == "exit":
        break

    try:
        result = eval(expression)
        print(f"النتيجة: {result}")
    except:
        print("عملية غير صحيحة، حاول مرة أخرى")

قائمة مهام بسيطة

برنامج لإضافة وعرض المهام اليومية.

tasks = []

while True:
    print("\n1. عرض المهام")
    print("2. إضافة مهمة")
    print("3. حذف مهمة")
    print("4. خروج")

    choice = input("اختر خيارًا: ")

    if choice == "1":
        if not tasks:
            print("لا توجد مهام!")
        else:
            for i, task in enumerate(tasks, 1):
                print(f"{i}. {task}")

    elif choice == "2":
        task = input("ادخل المهمة الجديدة: ")
        tasks.append(task)
        print("تمت إضافة المهمة!")

    elif choice == "3":
        if not tasks:
            print("لا توجد مهام للحذف!")
        else:
            task_num = int(input("ادخل رقم المهمة للحذف: ")) - 1
            if 0 <= task_num < len(tasks):
                del tasks[task_num]
                print("تم حذف المهمة!")
            else:
                print("رقم المهمة غير صحيح!")

    elif choice == "4":
        break

    else:
        print("خيار غير صحيح، حاول مرة أخرى")
